<p>I
 have written the below queries as I migrate my PHP website to the Yii2 
framework. I want to add them to my controller so as to display the top 
10 bets won.  I have tried going through many Yii2 database classes but I
 cannot get it to work.</p>

<p>My tables are:</p>

<p><strong>users:</strong></p>

<pre><code>id | user_name | user_status | ...other columns...
</code></pre>

<p><strong>bets:</strong></p>

<pre><code>id | user_id | date_time |...other columns...| balance_return
</code></pre>

<p>The queries I want to get in Yii2 are:</p>

<pre><code>$query_all = $dbh-&gt;query("
    SELECT SUM(bets.balance_return) AS total_win
         , bets.user_id
         , users.user_name
         , users.user_status
      FROM bets INNER JOIN users ON bets.user_id = users.id
     WHERE users.user_status = 'verified'
       AND bets.date_time &gt; " . $start_date . "
  GROUP BY bets.user_id
  ORDER BY total_win DESC
");
</code></pre>

<p>The variable start_date is a period of 6 months which I calculate according to <code>time()</code> Also please note that <code>balance_return</code> is every win a user got so its sum determines the ranking.</p>

<p>The second query is:</p>

<pre><code>$qwi = $dbh-&gt;query("
    SELECT SUM(bets.balance_return) AS total_win
         , bets.user_id
         , users.user_name
         , users.user_status
      FROM bets INNER JOIN users ON bets.user_id = users.id
     WHERE users.user_status = 'verified'
       AND bets.date_time &gt; " . $start_date . "
  GROUP BY bets.user_id
  ORDER BY total_win DESC LIMIT 0,10
");
</code></pre>

<p>You can execute raw sql like this</p>

<pre><code>$connection = Yii::$app-&gt;getDb();
$command = $connection-&gt;createCommand('
    SELECT SUM(bets.balance_return) AS total_win
     , bets.user_id
     , users.user_name
     , users.user_status
    FROM bets INNER JOIN users ON bets.user_id = users.id
    WHERE users.user_status = 'verified'
    AND bets.date_time &gt; :start_date
    GROUP BY bets.user_id
    ORDER BY total_win DESC', [':start_date' =&gt; '1970-01-01']);

$result = $command-&gt;queryAll();
</code></pre>

<p>I recommend reading: <a href="http://www.yiiframework.com/doc-2.0/yii-db-connection.html#createCommand()-detail">http://www.yiiframework.com/doc-2.0/yii-db-connection.html#createCommand()-detail</a> </p>

<p>The first parameter is the sql (with placeholder(s)) and the second part is an array of values to be used with the placeholders.</p>
